## 全局安装 Koa 脚手架

`$ npm install -g koa-generator`

## 查看帮助

`$ koa2 -h`

## 搭建 Koa 项目

`$ koa2 koa-demo`

默认使用 pug 模板引擎

## 安装依赖

`$ npm install`

## 启动本地调试环境

`$ npm start`

默认监听端口为 3000.
可在 http://localhost:3000 访问项目.
