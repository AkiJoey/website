## 查看版本

`$ npm -v`

## 安装依赖

`$ npm install`

`-S` 为 `--save` 简写, 写入 dependencies.
`-D` 为 `--save-dev` 简写, 写入 devDependencies.

## 更新依赖

`$ npm update`

## 卸载依赖

`$ npm uninstall`

## 查看依赖

`$ npm ls`

-g 查看全局依赖.
--depth=0 查看深度为 0 的依赖.

## 查看 npm 源

`$ npm config list`
