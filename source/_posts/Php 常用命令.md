## 查看版本

`$ php -v`

## 启动 php-fpm 服务

`$ php-fpm`

## 显示 php 配置

`$ php -i`

## 显示 php 扩展

`$ php -m`

## 命令行运行 php 代码

`$ php -r "phpinfo();"`

## 进入交互模式

`$ php -a`

`exit` 退出交互模式.

## 查看帮助

`$ php -h`
